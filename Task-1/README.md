Set up cloud storage using Amazon S3 by creating a bucket, uploading files, enabling versioning, and configuring bucket access and encryption.

- Upload sample objects.  
- Configure an appropriate bucket access policy.  
- Apply IAM least-privilege policy for access.  
- Enable server-side encryption (SSE).
